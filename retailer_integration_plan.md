**Retailer Integration Plan (Draft)**

**Scoring**
- P0: High impact, high spend flow, required for price comparison coverage.
- P1: High relevance; important for category depth or luxury coverage.
- P2: Regional or long-tail; add after P0/P1.

**Access Methods**
- Public API: official catalog endpoints.
- Partner/affiliate feed: batch or near-real-time feeds.
- Monitoring: site change tracking where permitted.

**Retailers**
- Amazon (US/CA) — P0 — Affiliate/Partner API (PA-API / Creators) + data partnership — Affiliate access required; product data may be limited by category policies.
- Walmart (US/CA) — P0 — Marketplace API — Official marketplace API for catalog, price, inventory.
- Target (US/CA) — P0 — No public catalog API; use affiliate/partner feeds or monitored extraction — Partner feed preferred; scraping only where permitted.
