## 📱 How to Use in Figma

### Method 1: Import SVG (Recommended)
1. Open Figma
2. Go to **File** → **Import**
3. Select one of the SVG files
4. The mockup will import as editable vectors
5. Delete the instruction box at the bottom
6. Select the placeholder rectangle in the phone
7. Go to **File** → **Place Image** (or Cmd+Shift+K)
8. Choose your Muse app screenshot
9. Adjust screenshot to fit the phone frame
10. Export as PNG (1242×2208px)

### Method 2: Drag & Drop
1. Open the SVG file in Preview/browser
2. Drag it directly into your Figma canvas
3. Follow steps 5-10 above

## 🎯 Tips for Best Results

1. **Take clean mobile screenshots** (414×896px or similar)
2. **Crop to show the best part** of each page
3. **Use Figma's mask feature** to fit screenshot in phone frame perfectly
4. **Match the phone screen size** (396×780px area inside the frame)
5. **Export at 2x** for retina quality (File → Export → 2x)
